# 文本清洗专家 (Text Cleaning Tool)

一个基于 Python (Tkinter) 开发的轻量级文本处理工具，专为AI写作和编辑设计。
它可以高效地清洗文本中的干扰符号（如星号、括号），并提供交互式的“误删恢复”功能，防止重要内容丢失。

## ✨ 功能特点

 智能清洗：一键去除文本中的星号 ``。
 括号分析：自动扫描文本中的 `()` 和 `（）` 内容，生成待删除列表。
 交互式恢复：
     在删除前，程序会列出所有检测到的括号内容。
     提供上下文对比（原文 vs 预期结果）。
     勾选“恢复”即可保留该段内容，未勾选的将自动删除。
 安全保护：分析期间自动锁定文本框，防止误触。
 字数统计：保存时自动显示处理后的“总字符数”和“纯字符数（去空）”。
 便捷操作：支持一键粘贴剪贴板内容、一键清空重置。
 格式保留：完美保留原文本的段落、换行格式，自动提取首行内容作为文件名。
